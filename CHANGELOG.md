# Changelog

## v1.0.5
- ✍️ 重写正负电提示文字显示：改为直接在世界坐标层渲染文字与背景，不再依赖 UI 层投影换算。
- 🧭 文字锚点固定在超速范围上方，并按相机缩放适配字号，解决“黑框有字无”的显示链路问题。

## v1.0.4
- 🧱 修复“只有黑框没有文字”的渲染问题：移除提示文字背景的错误图层设置，避免文字被背景遮挡。
- ✨ 保持多电网合并计算逻辑，优化显示稳定性与可读性。

## v1.0.3
- ⚖️ 调整正负电预判逻辑：当超速覆盖到多个电网时，不再取最差电网，改为将所有受影响电网的预测功率合并后再判断正/负。
- 🔎 这样在多电网交叠区域能反映整体效果，提示与实战体感一致。

## v1.0.2
- 🩹 修复正负电提示文字不显示：将世界投影坐标换算到 Scene/UI 坐标后再绘制，适配不同分辨率与 UI 缩放。
- ✅ 保留 v1.0.1 的稳定化改动，并补齐文本层显示链路，避免“有圈无字”。

## v1.0.1
- 🛠️ 修复正负电提示漂移：将文字层改为 `uiDrawEnd` 渲染，避免世界坐标与屏幕坐标混用导致的抖动/偏移。
- 🧹 移除无必要的小地图附着 overlay，解决提示层乱动和 HUD 干扰问题。

## v1.0.0
- ✨ 首发 BetterProjectorOverlay：手持超速投影/超速穹顶时显示放置后正负电预判。
- 🗺️ 新增超速冲突扫描：检测“超速投影位于超速穹顶或加布超速范围内”并调用 MindustryX 地图标记接口。
- 💬 新增聊天栏通知开关与坐标去重播报，消息格式为 `<BPO><应拆除超速>(x,y)`（中英适配）。
- 📦 补充 Release 流水线：按 tag 自动构建并发布 zip/jar/android.jar，并自动提取对应更新日志。
